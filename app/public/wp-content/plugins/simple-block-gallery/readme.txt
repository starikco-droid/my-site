Add the effect of Masonry and Slider to images.

== Description ==

= Only HTML & CSS =
* Create an HTML and CSS only block. The display does not collapse even if the plugin is deactivate.

= Masonry settings =
* Width
* Space
* Rounded Images

= Slider settings =
* Interval

= How it works =
[youtube https://youtu.be/aQsExk6eDhs]

== Installation ==

1. Upload `simple-block-gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
