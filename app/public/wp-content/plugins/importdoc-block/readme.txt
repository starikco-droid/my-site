Put the contents of a Google Doc in a WordPress post. When you update the Google Doc changes reflect on your website instantly.

== Description ==

This is a Block for the ImportDoc service. Simply signup at [importdoc.com](https://importdoc.com) and add a document, then paste the ImportDoc ID in the block settings.
